# Test Task — Frontend

## Реализовано
- Кастомная тройная шапка с бегущей строкой и бургер-меню
- Адаптивная навигация для desktop и mobile
- Универсальное модальное окно с разными сценариями открытия
- Валидация форм и защита от повторных действий
- Интерактивная витрина с фильтрацией и анимацией

## Особенности
- Одна модалка — разные сценарии
- Корректная логика сессии (исключение повторных модалок)
- Анимации и внимание к UX-мелочам
- Полностью адаптивная верстка
